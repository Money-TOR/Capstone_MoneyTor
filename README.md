# MoneyTOR
MoneyTOR adalah aplikasi manajemen keuangan dan operasional UMKM berbasis web yang membantu pencatatan transaksi, pengelolaan inventaris, visualisasi data bisnis, serta menyediakan AI Assistant berbasis Generative AI untuk mendukung pengambilan keputusan yang lebih cerdas.
